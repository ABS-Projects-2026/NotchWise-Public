# Contributing to NotchWise

NotchWise is proprietary software. The source code is not included in this repository and is not available for public contribution.

## What You Can Do

We welcome:

- **Bug reports** — Something not working? Open an [Issue](https://github.com/ABS-Projects-2026/NotchWise-Public/issues)
- **Feature requests** — Have an idea? We'd love to hear it via [Issues](https://github.com/ABS-Projects-2026/NotchWise-Public/issues)
- **Questions** — Not sure how something works? Ask in [Issues](https://github.com/ABS-Projects-2026/NotchWise-Public/issues) or email [support@notchwise.app](mailto:support@notchwise.app)

## Bug Reports

To help us fix issues quickly, please include:

- **macOS version** (e.g., macOS 15.3 Sequoia)
- **Mac model** (e.g., MacBook Pro 14" M3, iMac 24" M4)
- **NotchWise version** (found in the menu bar menu → About NotchWise)
- **Steps to reproduce** the issue
- **Expected behaviour** vs. **actual behaviour**
- **Screenshots or screen recordings** if applicable

## Feature Requests

When suggesting a feature, it helps to describe:

- What you're trying to accomplish
- How you currently work around the limitation
- How you imagine the feature working

## Security Issues

Please do **not** report security vulnerabilities via public Issues. See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.
